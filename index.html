⚡ One-time setup required — run the local proxy
Mintsoft's API blocks browser requests (CORS). A tiny local proxy fixes this in seconds:
  1. Download mintsoft-proxy.ps1 (link below) and save it anywhere on your PC
  2. Right-click it → Run with PowerShell  (or: open PowerShell, run .\mintsoft-proxy.ps1)
  3. Leave that window open — the proxy runs silently in the background
  4. Come back here, paste your API key, click Refresh Now
The proxy only runs on your machine and only forwards requests to api.mintsoft.co.uk. Nothing is stored or sent anywhere else.
Bake your API key in: run as .\mintsoft-proxy.ps1 -ApiKey "YOUR_KEY" and leave the API Key field in the dashboard blank.
